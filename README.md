# Rachuonyo Hospital Management System (HMS)

Rachuonyo HMS is a web-based Hospital Management System designed to simplify and automate hospital operations such as patient registration, login, records management, and basic administrative tasks.

## Features
- Demo login system  
- Patient records interface  
- Clean and responsive UI  
- Works on desktop and mobile browsers  

## Technologies Used
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
